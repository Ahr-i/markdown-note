# - Code Block

'`'를 사용하여 Code Block을 만들 수 있다.   
작은 따옴표가 아니라 왼쪽 상단에 있는 역따옴표다.   

작성 방법은 다음과 같다.

```
'''
#include <bits/stdc++.h>

using namespace std;

int main(int argc, char* argv[]) {
  cout << "Hello World!";

  return 0;
}
'''
```

```
#include <bits/stdc++.h>

using namespace std;

int main(int argc, char* argv[]) {
  cout << "Hello World!";

  return 0;
}
```

-  -  -

언어를 지정해줄 수도 있다.
```
'''C++
#include <bits/stdc++.h>

using namespace std;

int main(int argc, char* argv[]) {
  cout << "Hello World!";

  return 0;
}
'''
```

```C++
#include <bits/stdc++.h>

using namespace std;

int main(int argc, char* argv[]) {
  cout << "Hello World!";

  return 0;
}
```
